# Разработка комбинаций по диагонали

Рабочие материалы для создания комбинаций по диагонали зала.

## Цели работы по диагонали

- Развитие координации в пространстве
- Тренировка прыжков и вращений в движении
- Работа над баллоном и элевацией
- Развитие музыкальности и артистизма
- Подготовка к сценической практике

## Типы комбинаций

### Прыжковые комбинации
- Маленькие прыжки (temps leve, changement)
- Средние прыжки (jete, assemble)
- Большие прыжки (grand jete, saut de basque)
- Комбинированные прыжки

### Вращения
- Pirouette в движении
- Tour chaines
- Pique tour
- Fouette en tournant

### Комбинированные элементы
- Прыжки + вращения
- Шаги + прыжки
- Вращения + позы

## Принципы составления

1. **Постепенность** - начинать с простых элементов
2. **Безопасность** - учитывать физическую готовность
3. **Разнообразие** - чередовать типы движений
4. **Практичность** - готовить к сценической работе
5. **Музыкальность** - четкое соответствие музыке

## Советы

- Используйте всю длину диагонали
- Чередуйте правую и левую стороны
- Включайте подготовительные упражнения
- Записывайте успешные комбинации
- Адаптируйте под размеры зала
