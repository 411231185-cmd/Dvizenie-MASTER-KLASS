# Разработка комбинаций

Рабочие материалы для создания новых танцевальных комбинаций.

## Структура

### У станка
Разработки классических упражнений у станка

### По диагонали
Разработки прыжков, вращений и передвижений

### На середине класса
Разработки allegro, adagio, port de bras

### Этюды
Постановочные работы (этюды длительностью ~2 минуты)

## Формат описания комбинации

Создавайте файлы в формате Markdown: `[название]_комбинация.md`

### Шаблон описания комбинации

```markdown

**Уровень сложности:** [Начальный/Средний/Продвинутый]  
**Музыкальный размер:** [2/4, 3/4, 4/4, 6/8]  
**Темп:** [Медленно/Умеренно/Быстро]  
**Длительность:** [количество тактов/минут]

## Описание

[Краткое описание комбинации, цели, технические задачи]

## Музыкальное сопровождение

**Музыка:** [Композитор, произведение]  
**Файл:** [путь к музыкальному файлу]
```

## Рекомендации

- Записывайте комбинации сразу после их разработки
- Используйте стандартную хореографическую терминологию
- Указывайте музыкальное сопровождение
- Снимайте видео для наглядности
- Регулярно обновляйте и дорабатывайте комбинации
- Отмечайте уровень сложности для правильного подбора группе
