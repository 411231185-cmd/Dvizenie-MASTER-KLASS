# Старшая группа - Ансамбль хореографических миниатюр "Движение"

Материалы для проведения мастер-классов старшей группы.

## Структура папок

### Мастер-классы
Планы и материалы для проведения мастер-классов.

### Музыка
Музыкальное сопровождение для занятий и постановок.

### Видео
Видеозаписи выступлений и учебных материалов.

### Видео материалы
Организованная коллекция видео по типам комбинаций:
- **У станка** - упражнения у хореографического станка
- **По диагонали** - комбинации по диагонали зала
- **На середине класса** - танцевальные комбинации в центре зала
- **Этюды** - законченные танцевальные этюды (длительность ~2 минуты)

### Разработка комбинаций
Рабочие материалы для создания новых комбинаций:
- **У станка** - классические упражнения
- **По диагонали** - прыжки, вращения, передвижения
- **На середине класса** - allegro, adagio, port de bras
- **Этюды** - постановочные работы
