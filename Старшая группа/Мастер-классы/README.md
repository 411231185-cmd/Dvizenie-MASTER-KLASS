# Мастер-классы

## Структура мастер-класса

Каждый мастер-класс должен включать:

1. **Разминка** (10-15 минут)
   - Разогрев мышц
   - Подготовка тела к занятию

2. **Работа у станка** (20-25 минут)
   - Plie
   - Battement tendu
   - Battement jete
   - Rond de jambe
   - Battement fondu
   - Battement frappe
   - Grand battement

3. **Работа на середине класса** (20-25 минут)
   - Adagio
   - Port de bras
   - Allegro
   - Вращения

4. **Работа по диагонали** (15-20 минут)
   - Прыжки
   - Вращения
   - Шаги и комбинации

5. **Постановочная работа - Этюд** (15-20 минут)
   - Разучивание этюда на 2 минуты
   - Отработка техники
   - Работа над артистизмом

6. **Заминка и растяжка** (10 минут)
   - Упражнения на гибкость
   - Расслабление мышц

## Формат файла плана мастер-класса

Создавайте файлы в формате: `ГГГГ-ММ-ДД_тема_мастер-класса.md`

Пример: `2026-01-15_классический_танец.md`
