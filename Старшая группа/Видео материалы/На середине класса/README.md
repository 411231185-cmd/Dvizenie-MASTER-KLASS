# На середине класса - Комбинации

Видео материалы танцевальных комбинаций в центре зала.

## Типы комбинаций

### Adagio (медленные комбинации)
- Развитие устойчивости
- Работа над линиями
- Баланс и координация
- Плавность движений

### Port de bras (упражнения для рук и корпуса)
- Позиции рук
- Движения корпуса
- Координация рук и головы
- Выразительность

### Allegro (быстрые прыжки)
- Маленькие прыжки
- Средние прыжки
- Комбинации прыжков
- Баллон и элевация

### Вращения
- Подготовка к вращению
- Pirouette с разных позиций
- Многооборотные вращения
- Комбинации с вращениями

### Batterie (заноски)
- Entrechat
- Brise
- Royale

## Рекомендации по съемке

- Снимайте с фронта для видимости позиций
- Убедитесь, что видна вся комбинация
- Записывайте в медленном темпе для обучения
- Показывайте исполнение в рабочем темпе
