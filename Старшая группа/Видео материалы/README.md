# Видео материалы

Организованная коллекция видео по типам комбинаций для обучения и справки.

## Структура

### У станка
Видео упражнений у хореографического станка:
- Plie
- Battement tendu
- Battement jete
- Rond de jambe
- Battement fondu
- Battement frappe
- Grand battement

### По диагонали
Видео комбинаций по диагонали зала:
- Прыжки (soubresaut, changement, pas echappe)
- Вращения (pirouette, tour)
- Шаги и передвижения (pas de bourree, glissade)

### На середине класса
Видео танцевальных комбинаций в центре зала:
- Adagio (медленные комбинации)
- Allegro (быстрые прыжки)
- Port de bras (упражнения для рук и корпуса)
- Вращения

### Этюды
Законченные танцевальные этюды (длительность ~2 минуты):
- Классические этюды
- Современные этюды
- Народные этюды
- Характерные этюды

## Формат названий файлов

`[категория]_[название]_[дата].mp4`

Примеры:
- `станок_grand_battement_2026-01-15.mp4`
- `диагональ_pirouette_2026-01-20.mp4`
- `середина_adagio_2026-02-01.mp4`
- `этюд_зимняя_сказка_2026-02-10.mp4`

## Рекомендации

- Снимайте комбинации с разных ракурсов
- Записывайте как преподавателя, так и учеников
- Добавляйте пояснения в отдельный текстовый файл
- Группируйте видео по уровню сложности
