# Этюды

Законченные танцевальные этюды (длительность ~2 минуты).

## Структура этюда

Этюд - это небольшое законченное танцевальное произведение, которое включает:
- Четкое начало и окончание
- Развитие темы или образа
- Использование изученной техники
- Артистическую выразительность

## Типы этюдов

### Классические этюды
Основаны на технике классического танца

### Современные этюды
Используют современную хореографическую лексику

### Народные этюды
Включают элементы народного танца

### Характерные этюды
Передают определенный характер, настроение или образ

## Требования к этюдам

- **Длительность:** приблизительно 2 минуты
- **Техника:** соответствует уровню старшей группы
- **Музыкальность:** четкая связь с музыкой
- **Выразительность:** передача образа и эмоций
- **Композиция:** логичное построение

## Этапы работы над этюдом

1. **Прослушивание музыки** - выбор и анализ
2. **Разработка композиции** - структура этюда
3. **Создание хореографии** - постановка движений
4. **Разучивание** - изучение по частям
5. **Отработка техники** - работа над чистотой исполнения
6. **Работа над образом** - развитие артистизма
7. **Сводка** - объединение всех частей

## Формат названий файлов

`этюд_[название]_[стиль]_[дата].mp4`

Примеры:
- `этюд_зимняя_сказка_классический_2026-01-15.mp4`
- `этюд_осенний_вальс_современный_2026-02-01.mp4`

## Рекомендации по съемке

- Снимайте полный прогон этюда
- Записывайте с разных точек зала
- Сохраняйте как рабочие, так и итоговые версии
- Записывайте процесс разучивания для анализа
