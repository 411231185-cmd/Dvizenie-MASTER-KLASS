# У станка - Упражнения

Видео материалы упражнений у хореографического станка.

## Типы упражнений

1. **Plie** - приседания
2. **Battement tendu** - вытягивание ноги
3. **Battement tendu jete** - маленькие броски
4. **Rond de jambe par terre** - круговые движения ногой по полу
5. **Battement fondu** - тающие движения
6. **Battement frappe** - удары
7. **Rond de jambe en l'air** - круговые движения ногой в воздухе
8. **Petit battement** - маленькие удары
9. **Grand battement** - большие броски

## Рекомендации по съемке

- Снимайте как с правой, так и с левой ноги
- Показывайте упражнение целиком
- Включайте вид спереди и сбоку
- Записывайте музыкальное сопровождение
