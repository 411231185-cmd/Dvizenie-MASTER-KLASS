# По диагонали - Комбинации

Видео материалы комбинаций по диагонали зала.

## Типы комбинаций

### Прыжки
- **Temps leve** - прыжок на одной ноге
- **Soubresaut** - прыжок с двух на две
- **Changement** - прыжок со сменой ног
- **Pas echappe** - прыжок с раскрыванием ног
- **Pas assemble** - прыжок с собиранием ног
- **Sissonne** - прыжок с двух на одну
- **Jete** - бросок

### Вращения
- **Pirouette** - вращение на одной ноге
- **Tour chaines** - цепочка вращений
- **Pique tour** - вращение с шагом на полупальцы

### Шаги и передвижения
- **Pas de bourree** - переступание
- **Glissade** - скольжение
- **Pas de basque** - баскский шаг
- **Balance** - качание

## Рекомендации по съемке

- Снимайте движение по всей диагонали
- Убедитесь, что виден путь движения
- Записывайте несколько повторений
- Показывайте как правильное, так и типичные ошибки
