# Видео

Видеозаписи выступлений и учебных материалов.

## Организация файлов

Сохраняйте видео в следующих форматах:
- MP4 (рекомендуется)
- MOV
- AVI

## Структура названий файлов

Рекомендуемый формат: `ГГГГ-ММ-ДД_[тип]_[название].mp4`

Примеры:
- `2026-01-15_выступление_отчетный_концерт.mp4`
- `2026-01-20_урок_классический_танец.mp4`
- `2026-02-01_репетиция_этюд_весна.mp4`

## Категории

- **Выступления** - записи концертов и выступлений
- **Уроки** - записи мастер-классов и занятий
- **Репетиции** - рабочие видео репетиционного процесса
- **Обучающие** - учебные материалы, примеры техники

## Рекомендации

- Записывайте видео в горизонтальной ориентации
- Обеспечьте хорошее освещение
- Записывайте в максимально доступном качестве (минимум 720p)
- Делайте резервные копии важных записей
- Используйте стабилизацию или штатив
