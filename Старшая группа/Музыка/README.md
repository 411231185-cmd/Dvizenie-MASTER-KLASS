# Музыка

Музыкальное сопровождение для занятий и постановок.

## Организация файлов

Сохраняйте музыкальные файлы в следующих форматах:
- MP3
- WAV
- FLAC

## Структура названий файлов

Рекомендуемый формат: `[категория]_[название]_[темп].mp3`

Примеры:
- `станок_plie_медленно.mp3`
- `середина_allegro_быстро.mp3`
- `этюд_лебединое_озеро_отрывок.mp3`

## Категории

- **Станок** - музыка для упражнений у станка
- **Середина** - музыка для работы на середине класса
- **Диагональ** - музыка для диагональных комбинаций
- **Этюды** - музыка для постановочных работ
- **Разминка** - музыка для разминки
- **Растяжка** - спокойная музыка для заминки

## Рекомендации

- Используйте музыку с четким ритмом для технических упражнений
- Выбирайте выразительную музыку для этюдов
- Учитывайте музыкальный размер (2/4, 3/4, 4/4, 6/8)
- Сохраняйте оригинальные файлы в высоком качестве
